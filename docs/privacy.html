Coach Tarot

Politique de confidentialité

Dernière mise à jour : 4 juin 2026

En résumé : Coach Tarot fonctionne entièrement hors ligne. Toutes vos données de jeu (joueurs, sessions, scores) restent sur votre appareil. Nous ne collectons, ne stockons et ne transmettons aucune donnée personnelle à des serveurs distants. Le seul tiers ayant accès à certaines données techniques est Google AdMob, utilisé pour afficher les bannières publicitaires (sauf si vous avez acheté la version sans publicité).

1. Qui sommes-nous ?

Coach Tarot est une application mobile éditée par Gregre, développeur indépendant établi en France. Cette politique de confidentialité explique comment l'application traite les informations qui vous concernent.

2. Données stockées localement sur votre appareil

L'application enregistre les informations suivantes uniquement dans la mémoire interne de votre téléphone, via le mécanisme SharedPreferences d'Android :

  • Les noms et couleurs des joueurs de votre carnet
  • L'historique de vos sessions et des donnes jouées
  • Vos préférences (thème clair/sombre)
  • Le statut "sans publicité" si vous avez effectué l'achat correspondant

Ces données ne quittent jamais votre appareil. Elles sont supprimées dès que vous désinstallez l'application ou videz son cache via les paramètres Android.

3. Publicité (Google AdMob)

La version gratuite de Coach Tarot affiche des bannières publicitaires fournies par Google AdMob. Pour personnaliser ces publicités, Google peut collecter et traiter :

  • Votre identifiant publicitaire (Advertising ID)
  • Des informations techniques sur votre appareil (modèle, OS, langue)
  • Des données d'utilisation des publicités (impressions, clics)

Vous pouvez réinitialiser votre identifiant publicitaire ou désactiver la personnalisation depuis les paramètres Android : Paramètres → Confidentialité → Annonces.

Les pratiques de Google sont régies par leur propre politique :
https://policies.google.com/technologies/partner-sites

4. Achat intégré "Sans publicité"

Coach Tarot propose un achat unique (~2,49 €) pour supprimer définitivement les publicités. Cette transaction est gérée intégralement par Google Play Billing. Nous ne recevons et ne stockons aucune coordonnée bancaire. Seul un indicateur "achat effectué" est enregistré localement sur votre appareil pour masquer les publicités.

5. Partage et import de sessions

L'application vous permet d'exporter vos sessions au format JSON et de les partager via les applications de votre choix (e-mail, messagerie, cloud personnel). De même, vous pouvez importer un fichier JSON précédemment exporté. Ces actions sont entièrement à votre initiative : aucune donnée n'est partagée automatiquement.

6. Permissions Android

L'application demande uniquement les permissions strictement nécessaires :

  • Internet — uniquement pour charger les bannières publicitaires AdMob (non utilisée si vous êtes Premium)
  • Accès aux fichiers — uniquement déclenché par vous, au moment de l'import/export d'une session JSON

L'application ne demande aucune autre permission sensible (caméra, contacts, géolocalisation, etc.).

7. Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données, vous disposez d'un droit d'accès, de rectification, d'effacement et de portabilité de vos données. Étant donné que toutes vos données restent sur votre appareil :

  • Accès et portabilité : utilisez la fonction "Exporter en JSON" intégrée à l'application
  • Effacement : désinstallez l'application ou supprimez ses données depuis les paramètres Android

Pour les données traitées par Google AdMob, exercez vos droits directement auprès de Google via les liens fournis dans leur politique.

8. Enfants

Coach Tarot s'adresse à un public de joueurs adultes (18+). Nous ne collectons délibérément aucune information sur des mineurs.

9. Modifications de cette politique

Cette politique peut être mise à jour pour refléter les évolutions de l'application ou de la réglementation. La date de dernière mise à jour indiquée en haut de cette page fait foi. Les modifications majeures vous seront signalées via une notification dans l'application.
