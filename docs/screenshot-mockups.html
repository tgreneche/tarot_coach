📱 Mockups Screenshots Play Store

6 emplacements préparés avec cadre téléphone (1080 × 1920 portrait standard). Une fois rempli avec tes captures d'app réelles, screenshot chaque "mock" individuel pour obtenir 6 visuels propres à uploader dans Play Console.

Comment remplir avec tes vraies captures :
  1. Lance l'app sur un émulateur Android (Android Studio → Device Manager → Pixel 7 → bouton ▶) ou sur un device réel branché en USB.
  2. Pour chaque écran clé : screenshot via Ctrl+S dans l'émulateur (ou Power + Vol- sur device).
  3. Place les fichiers PNG dans docs/screenshots/ avec ces noms exacts :
    01-home.png, 02-analyse-main.png, 03-selection-cartes.png, 04-session-board.png, 05-stats-joueurs.png, 06-recap.png
  4. Rouvre cette page : les images apparaissent dans les frames automatiquement.
  5. Pour uploader sur Play Store : tu peux soit (a) uploader directement tes captures brutes en PNG dans Play Console (le plus simple), (b) ou screenshot chaque mock framé d'ici pour un rendu plus stylé.
